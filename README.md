# TALENT UI React

React + Vite implementation based on the uploaded TALENT UI PDF.

## Run locally

```bash
npm install
npm run dev
```

## Routes

### Public
- /
- /about
- /products
- /resources
- /distributors
- /contact

### Admin
- /admin/dashboard
- /admin/products
- /admin/products/new
- /admin/resources
- /admin/inquiries
- /admin/distributors
- /admin/analytics
- /admin/users-roles
- /admin/settings

## Notes
- The layout, section order, and visual style follow the uploaded UI document.
- Placeholder images are used for manufacturing visuals because the PDF does not include exportable original assets.
- Static mock data is used to mirror the admin screens.
